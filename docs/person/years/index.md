# 年

过年的时刻，对我而言反而是一年中最悲伤的时刻。一方面是总要见一个永远长不大的父亲，另一方面则是心疼渐渐老去的老妈。<br>

我成长的速度远远没有跟上老妈老去的速度，每每见到老妈老去的皱纹和忙碌的身影，我都会对自己的无能感到愤怒。<br>

如果有人问我，谁是世界上最伟大的人，那我一定会回答是我母亲。我母亲是个很勤劳老实的人，从我有记忆起，父亲这个词就离我很遥远，这个词在我成长期间几乎是消失的，哪怕出现，也只是又听到欠谁谁谁一笔钱，我讨厌自己的父亲。而母亲则是总是见到忙碌的背影，我母亲是一个很好很好的人，舍弃了自己的生活，舍弃了自己的脾气，舍弃了许许多多，忙碌了半辈子终于将我和我姐带大。<br>

我是负二代，小时候也因家境贫寒而感到自卑，小时候，我很讨厌家里开的小店，因为同学会嘲笑我家穷，没少受屈辱。但现在我则是为小店感到自豪，因为这是我母亲开的，这就是我的家。如果有机会，我很想向曾经嘲笑过我的同学介绍一下我的母亲，介绍一下我家的小店，对了，我家的炒牛肉真的很好吃。<br>

写这篇文章的时候，我是哭的，我很少哭。看着长辈眼角的皱纹，我会更加的厌恶无能的自己。很多时候我都可以做的更好，但却选择了沉溺于多巴胺的分泌和及时行乐的满足，而结果就是过年时见到长辈后的羞愧与伤感。<br>

接触过群山，才会明白曾经所处的不过是一个土丘。若未见阳光，本可忍受黑暗。新的一年，我也要成为山，成为光，才能赶上长辈老去的速度。<br>
